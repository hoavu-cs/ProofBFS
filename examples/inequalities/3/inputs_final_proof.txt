THEOREM
======================================================================
Prove sqrt(a^2 + 1/b^2) + sqrt(b^2 + 1/c^2) + sqrt(c^2 + 1/a^2) >= sqrt(97)/2.
(Problem 7)

SETUP
----------------------------------------------------------------------
  Let a, b, c be positive real numbers such that a + b + c <= 2
  [FACT] AM-GM inequality: for positive reals x, y, x + y >= 2*sqrt(x*y), equivalently sqrt(x*y) <= (x+y)/2
  [FACT] Cauchy-Schwarz in the form: (sum_i u_i * v_i)^2 <= (sum_i u_i^2)(sum_i v_i^2)

PROOF
----------------------------------------------------------------------
Step 1. For any positive real numbers a, b, c,
√(a² + 1/b²) + √(b² + 1/c²) + √(c² + 1/a²) ≥ √((a+b+c)² + (1/b + 1/c + 1/a)²).

  Proof: Set v₁ = (a, 1/b), v₂ = (b, 1/c), v₃ = (c, 1/a) and let S = ||v₁|| + ||v₂|| + ||v₃||, where ||·|| is the Euclidean norm. Then
S² = (||v₁|| + ||v₂|| + ||v₃||)² = ∑_{i=1}³ ||v_i||² + 2∑_{1≤i<j≤3} ||v_i||·||v_j||.
For any i, j the Cauchy‑Schwarz inequality gives v_i·v_j ≤ ||v_i||·||v_j||, i.e. ||v_i||·||v_j|| ≥ v_i·v_j.
Applying this to each pair,
S² ≥ ∑||v_i||² + 2∑_{i<j} v_i·v_j = (∑ a_i)² + (∑ 1/b_i)² = (a+b+c)² + (1/b + 1/c + 1/a)².
Since S ≥ 0, taking square roots yields S ≥ √((a+b+c)² + (1/b + 1/c + 1/a)²), which is the stated inequality.

Step 2. For positive real numbers a, b, c satisfying a + b + c ≤ 2,
(a+b+c)² + (1/a + 1/b + 1/c)² ≥ 97/4.

  Proof: Set t = a+b+c, so 0 < t ≤ 2. By Cauchy‑Schwarz,
(a+b+c)(1/a + 1/b + 1/c) ≥ (1+1+1)² = 9,
hence 1/a + 1/b + 1/c ≥ 9/t.
Therefore
(a+b+c)² + (1/a + 1/b + 1/c)² ≥ t² + (9/t)² = t² + 81/t².
Define f(t) = t² + 81/t². Multiply the desired inequality f(t) ≥ 97/4 by 4t² > 0:
4t⁴ + 324 ≥ 97t² ⇔ 4t⁴ − 97t² + 324 ≥ 0.
Put u = t² ∈ (0,4]; the inequality becomes 4u² − 97u + 324 ≥ 0.
The quadratic 4u² − 97u + 324 has discriminant 4225 = 65² and roots u = 4, u = 20.25. Because the leading coefficient is positive, the quadratic is non‑negative for u ≤ 4 (and for u ≥ 20.25). Since u ≤ 4, we have 4u² − 97u + 324 ≥ 0, i.e., 4t⁴ − 97t² + 324 ≥ 0, which is equivalent to f(t) ≥ 97/4. Thus the original inequality holds.

Step 3. For positive real numbers a, b, c satisfying a + b + c ≤ 2,
√(a² + 1/b²) + √(b² + 1/c²) + √(c² + 1/a²) ≥ √(97)/2.

  Proof: From the previously verified results:
1. For any positive a, b, c,
√(a² + 1/b²) + √(b² + 1/c²) + √(c² + 1/a²) ≥ √((a+b+c)² + (1/a + 1/b + 1/c)²).
2. Under the extra condition a + b + c ≤ 2,
(a+b+c)² + (1/a + 1/b + 1/c)² ≥ 97/4.
Taking square roots of the inequality in 2 (all terms are non‑negative) gives
√((a+b+c)² + (1/a + 1/b + 1/c)²) ≥ √(97/4) = √(97)/2.
Combining 1 and this bound yields the desired inequality.

QED
